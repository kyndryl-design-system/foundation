## Usage

### SCSS Mixins

```css
@use '@kyndryl-design-system/shidoka-foundation/scss/mixins/motion.scss';
$transitionPropsVar: background-color, border;
$duration: var(--kd-anim-duration-med);
$easing: var(--kd-anim-easing-in);

.your-css-selector {
  @include motion.transition($transitionPropsVar);
}

.your-css-selector {
  @include motion.transition($transitionPropsVar, $duration);
}

.your-css-selector {
  @include motion.transition($transitionPropsVar, $duration, $easing);
}
```
