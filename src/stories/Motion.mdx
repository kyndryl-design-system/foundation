# Usage

### SCSS Mixins

```css
@use '@kyndryl-design-system/shidoka-foundation/scss/mixins/motion.scss';
$transitionPropsVar: background-color, border;

.your-css-selector {
  @include motion.transition(
    $transitionPropsVar,
    var(--kd-anim-duration-med),
    var(--kd-anim-easing-in)
  );
}

.your-css-selector {
  @include motion.transition($transitionPropsVar, var(--kd-anim-duration-med));
}

.your-css-selector {
  @include motion.transition($transitionPropsVar);
}

.your-css-selector {
  @include motion.transition(
    $transitionPropsVar,
    150ms,
    cubic-bezier(0.1, 0.7, 1, 0.1)
  );
}
```

### CSS Variables

```css
.your-css-selector {
  transition: background var(--kd-anim-duration-med) var(--kd-anim-easing-in), border
      var(--kd-anim-duration-short) var(--kd-anim-easing-linear);
}
```

#

## Variables/Tokens

| Duration                  | Value |
| ------------------------- | ----- |
| --kd-anim-duration-short  | 150ms |
| --kd-anim-duration-med    | 300ms |
| --kd-anim-duration-long   | 400ms |
| --kd-anim-duration-x-long | 400ms |

| Easing                  | Value       |
| ----------------------- | ----------- |
| --kd-anim-easing-linear | linear      |
| --kd-anim-easing-in     | ease-in     |
| --kd-anim-easing-out    | ease-out    |
| --kd-anim-easing-in-out | ease-in-out |

# Usage

### SCSS Mixins

```css
@use '@kyndryl-design-system/shidoka-foundation/scss/mixins/motion.scss';

@keyframes animation1 {
  from {
    border: none;
  }
  to {
    border: 2px solid var(--kd-color-border-primary);
  }
}

.animation-1 {
  @include motion.animation(
    animation1,
    var(--kd-anim-duration-med),
    var(--kd-anim-easing-in)
  );
}

.your-css-selector {
  @include motion.animation(animation1, var(--kd-anim-duration-med));
}

.your-css-selector {
  @include motion.animation(animation1);
}

.your-css-selector {
  @include motion.animation(animation1, 150ms, cubic-bezier(0.1, 0.7, 1, 0.1));
}
```

### CSS Variables

```css
@keyframes animation2 {
  from {
    color: var(--kd-color-text-primary);
  }
  to {
    color: var(--kd-color-text-success);
  }
}

.your-css-selector {
  animation: animation2 var(--kd-anim-duration-med) var(--kd-anim-easing-in);
}
```
