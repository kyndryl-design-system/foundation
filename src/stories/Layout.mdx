# Layout

Work in progress

## Usage

### SCSS Mixins

```css
@use '~@kyndryl-design-system/foundation/scss/mixins/layout.scss';

.your-css-selector {
  @include layout.dark-theme-darkstone-80;
}
```

### Utility Classes

#### Import to SCSS

```css
@use '~@kyndryl-design-system/foundation/scss/utility/layout.scss';
```

#### Import to CSS

```css
@import '@kyndryl-design-system/foundation/css/layout.css';
```

#### Import to JS

```js
import '@kyndryl-design-system/foundation/css/layout.css';
```

#### Example HTML

```html
<html class="kd-theme--darkstone-80">
  <!-- rest of your HTML here -->
</html>
```
