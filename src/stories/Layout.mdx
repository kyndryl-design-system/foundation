# Layout

Work in progress

## Usage

### SCSS Mixins

```css
@use '~@kyndryl-design-system/foundation/scss/mixins/layout.scss';

.your-css-selector {
  @include layout.aspect-ratio('16:9');
}

.your-css-selector img {
  @include layout.object-fit('cover');
}
```

### Utility Classes

#### Import to SCSS

```css
@use '~@kyndryl-design-system/foundation/scss/utility/layout.scss';
```

#### Import to CSS

```css
@import '@kyndryl-design-system/foundation/css/layout.css';
```

#### Import to JS

```js
import '@kyndryl-design-system/foundation/css/layout.css';
```

#### Example HTML

```html
<div class="kd-layout--aspect-ratio-16x9">
  <img src="your-image" class="kd-layout--object-fit-cover" />
</div>
```
