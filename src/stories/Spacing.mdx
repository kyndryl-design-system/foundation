## Usage

### SCSS Mixins

```css
@use '~@kyndryl-design-system/foundation/scss/mixins/spacing.scss';

.your-css-selector {
  @include spacing.spacer('margin', 'left', '48');
}
```

### Utility Classes

#### Import to SCSS

```css
@use '~@kyndryl-design-system/foundation/scss/utility/spacing.scss';
```

#### Import to CSS

```css
@import '@kyndryl-design-system/foundation/css/spacing.css';
```

#### Import to JS

```js
import '@kyndryl-design-system/foundation/css/spacing.css';
```

#### Example HTML

```html
<div class="kd-spacing--padding-right-2">Hello!</div>
```

#### Column class name schema

```
kd-spacing--${property}-right-${value}


```
