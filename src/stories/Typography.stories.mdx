## Usage

<br />

### SCSS Mixins

```css
@use '@kyndryl-design-system/foundation/scss/mixins/typography.scss';

.your-css-selector {
  @include typography.type-hero-01;
}
```

### Utility Classes

#### Import to JS

```js
import '@kyndryl-design-system/foundation/scss/utility/typography.scss';
// or
import '@kyndryl-design-system/foundation/css/typography.css';
```

#### Import to SCSS

```css
@use '@kyndryl-design-system/foundation/scss/utility/typography.scss';
```

#### Example HTML

```html
<h1 class="kd-type--hero-0i kd-type-weight-ultralight">Hero-01 Ultralight</h1>
```
