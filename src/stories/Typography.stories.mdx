## Usage

<br />

### SCSS Mixins

```css
@use '@kyndryl-design-system/web-components/scss/mixins/typography.scss';

.your-css-selector {
  @include typography.type-hero--01;
}
```

### Utility Classes

#### Import to JS

```js
import '@kyndryl-design-system/web-components/scss/utility/typography.scss';
// or
import '@kyndryl-design-system/web-components/css/typography.css';
```

#### Import to SCSS

```css
@use '@kyndryl-design-system/web-components/scss/utility/typography.scss';
```

#### Example HTML

```html
<h1 class="kyn-type--hero-01">Hero-01</h1>
```
