# Typography

## Examples

<Unstyled>
  <h1 className="kyn-type--head-1">Heading 1</h1>

<br />

## Usage

<br />

### SCSS Mixins

```css
@import '@kyndryl-design-system/web-components/scss/typography.scss';

.your-heading-css-selector {
  @include type-head-1;
}
```

### Utility Classes

```js
import '@kyndryl-design-system/web-components/css/typography.css';
```

```html
<h1 class="kyn-type--head-1">Heading 1</h1>
```

## Styles

| Style     | Mixin       | Utility Class    |
| --------- | ----------- | ---------------- |
| Heading 1 | type-head-1 | kyn-type--head-1 |
| Body 1    | type-body-1 | kyn-type--body-1 |
