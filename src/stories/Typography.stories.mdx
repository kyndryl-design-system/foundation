## Usage

<br />

### SCSS Mixins

```css
@import '@kyndryl-design-system/web-components/scss/typography.scss';

.your-heading-css-selector {
  @include type-head-1;
}
```

### Utility Classes

```js
import '@kyndryl-design-system/web-components/css/typography.css';
```

```html
<h1 class="type--head-1">Heading 1</h1>
```
