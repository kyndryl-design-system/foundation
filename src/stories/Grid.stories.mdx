## Usage

<br />

### SCSS Mixins

```css
@use '~@kyndryl-design-system/foundation/scss/mixins/grid.scss';

.your-container-css-selector {
  @include grid-container(
    $align-left: false,
    $no-max: false,
    $no-padding: false
  );
}

.your-child-css-selector {
  @include grid-col(num_cols);
}
```

### Utility Classes

#### Import to SCSS

```css
@use '~@kyndryl-design-system/foundation/scss/utility/grid.scss';
```

#### Import to CSS

```css
@import '@kyndryl-design-system/foundation/css/grid.css';
```

#### Import to JS

```js
import '@kyndryl-design-system/foundation/css/grid.css';
```

#### Example HTML

```html
<div class="kd-grid-container">
  <div class="kd-grid-col--sm-4 kd-grid-col--md-8 kd-grid-col--lg-12">
    12 cols @ max, xl, lg, 8 cols @ md, and 4 cols @ sm
  </div>
</div>
```

#### Column class name schema

```
kd-grid-col--${breakpoint}-${num_cols}
```

## Breakpoints

| Breakpoint   | Value (rem/px) | Columns | Gap  |
| ------------ | -------------- | ------- | ---- |
| Small (sm)   | 20 / 320       | 4       | 32px |
| Medium (md)  | 42 / 672       | 8       | 32px |
| Large (lg)   | 74 / 1184      | 12      | 32px |
| X-Large (xl) | 82 / 1312      | 12      | 32px |
| Max (max)    | 99 / 1584      | 12      | 32px |

## Additional Utility Classes

### Container

| Description                             | Class                         |
| --------------------------------------- | ----------------------------- |
| Left align grid container               | kd-grid-container--align-left |
| Remove grid container max-width         | kd-grid-container--no-max     |
| Remove grid container left/rightpadding | kd-grid-container--no-padding |
