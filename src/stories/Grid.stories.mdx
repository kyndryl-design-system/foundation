## Usage

<br />

### SCSS Mixins

```css
@use '@kyndryl-design-system/web-components/scss/mixins/grid.scss';

.your-container-css-selector {
  @include grid-container;
}

.your-child-css-selector {
  @include grid-col(num_cols);
}
```

### Utility Classes

#### Import to JS

```js
import '@kyndryl-design-system/web-components/scss/utility/grid.scss';
// or
import '@kyndryl-design-system/web-components/css/grid.css';
```

#### Import to SCSS

```css
@use '@kyndryl-design-system/web-components/scss/utility/grid.scss';
```

#### Example HTML

```html
<div class="kyn-grid--container">
  <div class="kyn-grid-sm-col--4 kyn-grid-md-col--8 kyn-grid-lg-col--12">
    12 cols @ max, xl, lg, 8 cols @ md, and 4 cols @ sm
  </div>
</div>
```

#### Column class name schema

```
kyn-grid-${breakpoint}-col--${num_cols}
```

## Breakpoints

| Breakpoint   | Value (rem/px) | Columns | Gap  |
| ------------ | -------------- | ------- | ---- |
| Small (sm)   | 20 / 320       | 4       | 32px |
| Medium (md)  | 42 / 672       | 8       | 32px |
| Large (lg)   | 66 / 1056      | 12      | 32px |
| X-Large (xl) | 82 / 1312      | 12      | 32px |
| Max (max)    | 99 / 1584      | 12      | 32px |
