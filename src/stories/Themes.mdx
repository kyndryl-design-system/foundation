# Themes

Use the theme switcher at the top of Storybook (picture icon) to switch between different themes. This can be done anywhere in the Storybook.

## Example

<Canvas>
  <Story of={ThemesStories.Example} />
</Canvas>

## Usage

### SCSS Mixins

```css
@use '~@kyndryl-design-system/foundation/scss/mixins/themes.scss';

.your-css-selector {
  @include themes.darkstone-80;
}
```

### Utility Classes

#### Import to SCSS

```css
@use '~@kyndryl-design-system/foundation/scss/utility/themes.scss';
```

#### Import to CSS

```css
@import '@kyndryl-design-system/foundation/css/themes.css';
```

#### Import to JS

```js
import '@kyndryl-design-system/foundation/css/themes.css';
```

#### Example HTML

```html
<body class="kd-theme--darkstone-80">
  <!-- rest of your HTML here -->
</body>
```
