## Contributing

### Scripts

```bash
# install
npm i --legacy-peer-deps

# run development env
npm run storybook

# build
npm run build

# lint
npm run lint

# test (must run build first)
npm run test

# generate new custom-elements.json, updates storybook docs/args
npm run analyze

# format all files with prettier
npm run format
```
